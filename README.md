# ❌⭕ Tic-Tac-Toe Web Game

The **Tic-Tac-Toe** is a classic 2-player turn-based game developed using HTML, CSS, and JavaScript. This project was built during my internship to enhance my logic-building and frontend interactivity skills.

---

## 📌 Project Description

Tic-Tac-Toe is a simple but strategic game where two players alternate placing Xs and Os in a 3×3 grid. The first player to align three of their marks horizontally, vertically, or diagonally wins the game.

---

## 🚀 Features

- 🎮 2-player turn-based gameplay
- ❌⭕ Dynamic "X" and "O" placement
- 🏆 Win detection (horizontal, vertical, diagonal)
- 🚫 Draw condition detection
- 🔄 Restart/Reset Game button
- 🎨 Interactive and responsive UI

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
